Plan de Capacitación: Despliegue del Sistema Multi-Agente MDS Formatter Builder

Público objetivo: Equipo de IT de MDS, ingenieros de operaciones, desarrolladores de formatters y personal técnico involucrado en el procesamiento de Collection Upload
Objetivo general: Capacitar al equipo de MDS para desplegar, operar, verificar y mantener el sistema MDS Formatter Builder, que utiliza agentes de IA especializados para la generación automática de formatters de carga de colecciones médicas, asegurando alineación con estándares MDS y compliance PHI.

Módulos

1. Introducción al Sistema MDS Formatter Builder y Arquitectura Multi-Agente

Objetivo: Comprender el contexto de negocio, la arquitectura del sistema, el rol de cada agente de IA y cómo reemplazan el flujo de trabajo manual actual.

  • Contexto de negocio: Procesamiento de Collection Upload en MDS (proveedores médicos, tipos de archivo: Meditech, Epic, Cerner)
  • Problema actual: Generación manual de formatters (PriXXXX.cs) y sus desafíos
  • Visión general del sistema: De un solo agente a arquitectura multi-agente especializada
  • Los 4 agentes principales y sus responsabilidades:
    - Match Specs: Auditor técnico de especificaciones
    - File Analyzer: Analizador de estructura de archivos
    - Code Generator: Generador de código C# con templates
    - Code Validator: Validador de compilación y estructura
    - Issue Resolver: Resolución interactiva de problemas
  • Fases del proceso: Análisis → Generación → Validación → Resolución Interactiva → Entrega
  • Beneficios esperados: Velocidad, consistencia, reducción de errores, alineación con estándares MDS

Duración: 1.5 horas

Metodología: Presentación con diagramas del flujo (UML), demostración del sistema en ambiente de prueba, discusión de casos de uso reales de MDS.
